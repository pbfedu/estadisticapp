Més informació sobre les Variables

Quan fem un estudi estadístic, és important saber quines variables volem mesurar. Hi ha tres tipus principals de variables:

  • Variables Quantitatives Discretes: Són aquelles que podem comptar i que tenen valors específics. Per exemple, el nombre de persones en una família o el nombre d'animals d'una granja. Aquestes variables no poden tenir valors decimals.
  • Variables Quantitatives Contínues: Són aquelles que poden tenir qualsevol valor dins d'un rang. Per exemple, l'altura o el pes d'una persona, que poden ser valors decimals com 1,75 metres o 70,5 kg. Aquestes variables poden ser mesurades amb molta precisió.
  • Variables Qualitatives: Són variables que no es poden mesurar numèricament, sinó que descriuen qualitats o característiques. Per exemple, el color d'ulls (blau, verd, marró) o el tipus de treball que fa una persona (metge, professor, enginyer). Aquestes variables es classifiquen en categories.
